Tehnologii Utilizate

React Framework

Biblioteca JavaScript pentru interfețe utilizator interactive, oferind componente refolosibile pentru dashboard și vizualizare criptografică.

Avantaje: UI reactiv, performanță, componentizare

D3.js

Bibliotecă pentru vizualizări de date, utilizată pentru reprezentarea grafică a analizelor criptografice.

Avantaje: Vizualizări complexe, manipulare SVG, interactivitate

CLI (Node.js)

Interfață linie de comandă construită pe Node.js pentru acces scriptabil la toate funcționalitățile.

Avantaje: Automatizare, integrare în pipeline-uri, utilizare avansată

Nginx

Server web pentru servirea aplicației și load balancing.

Avantaje: Performanță ridicată, configurare flexibilă, securitate

OpenSSL

Bibliotecă criptografică robustă pentru implementarea protocoalelor TLS/SSL și funcționalități criptografice de bază.

Avantaje: Suport algoritmi multipli, performanță ridicată, constant actualizată cu standardele moderne

Python (FastAPI)

Framework Python modern pentru API-uri, utilizat pentru orchestrarea componentelor și procesare.

Avantaje: Async, documentare automată, tipare, performanță

C/C++

Module de performanță pentru algoritmi criptografici intensivi.

Avantaje: Viteză maximă pentru operații criptografice complexe

Docker

Containerizare pentru toate serviciile platformei, facilitând deployment și scalare.

Avantaje: Izolare, reproducere, orchestrare

RsaCtfTool

Instrument specializat pentru atacuri asupra RSA, integrat cu API-uri.

Funcționalități: Factorizare, atac Wiener, recuperare chei

AES-CTF-Tool

Analiză și atac pentru criptogramele AES.

Funcționalități: Detecție mod, atac pad oracle, analiză entropică

CyberChef

Procesare multi-format cu operații în lanț, integrat pentru transformări.

Funcționalități: Conversii, encoding/decoding, analiză

Hash-Identifier

Modul de identificare automată a tipurilor de hash.

Funcționalități: Recunoaștere peste 40 tipuri, sugestii cracking

Blockchain Explorer

Instrument specializat pentru analiza tranzacțiilor blockchain și verificarea semnăturilor criptografice.

Funcționalități: Analiza semnăturilor ECDSA, urmărire tranzacții, verificare hash-uri block

TensorFlow/PyTorch

Framework-uri ML pentru analiza pattern-urilor și inferențe criptografice.

Aplicații: Clasificare criptosisteme, analiza pattern-urilor

Analiză de securitate a parolelor

Module AI pentru evaluarea robustețea parolelor și predicție vulnerabilități.

Metode: NLP, rețele neuronale, modele probabilistice

Detecție anomalii criptografice

Sisteme AI pentru identificarea implementărilor vulnerabile și riscurilor de securitate.

Tehnici: Învățare nesupervizată, clustering, auto-encodere

PostgreSQL

Bază de date relațională pentru stocarea datelor structurate ale platformei.

Utilizare: Utilizatori, proiecte, rezultate analize

MongoDB

Bază de date NoSQL pentru stocarea flexibilă a datelor semi-structurate.

Utilizare: Stocarea rezultatelor de analiză, date nestructurate

DB Specializată numere prime

Bază de date optimizată pentru stocarea și căutarea eficientă a numerelor prime și factorizări.

Optimizări: Indexare specială, algoritmi de căutare

Rainbow Tables

Bază de date specializată pentru hash-uri pre-calculate pentru reverse hashing.

Caracteristici: Compresie, indexare avansată, partajare
